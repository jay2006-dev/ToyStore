🧸 ToyStore – Full-Stack Web Application

ToyStore is a full-stack web application that simulates an online toy marketplace, built to demonstrate modern web development practices. The application allows users to browse, manage, and interact with toy products through a clean, responsive interface backed by robust backend logic and RESTful APIs.

This project emphasizes scalability, maintainability, and user-centric design while implementing real-world e-commerce workflows such as product listing, data management, and interactive UI components.

🚀 Features
🛍️ Product Browsing
Explore a collection of toys with an intuitive and responsive interface.
✏️ CRUD Operations
Create, read, update, and delete toy listings efficiently.
🔍 Search & Filtering
Quickly find products using optimized search and filtering functionality.
⚡ REST API Integration
Structured backend communication using RESTful endpoints.
📱 Responsive Design
Fully optimized for desktop and mobile devices.
🧩 Modular Architecture
Clean separation of frontend and backend for better scalability.

🎯 Use Case

ToyStore serves as a practical demonstration of how an e-commerce platform operates at a fundamental level. It can be extended with features like authentication, cart management, payment integration, and admin dashboards, making it a strong foundation for more advanced applications.
